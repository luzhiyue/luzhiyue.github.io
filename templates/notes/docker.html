Docker 备忘录

  • docker --version - 查看 Docker 版本
  • docker pull [image] - 从 Docker Hub 拉取镜像
  • docker images - 列出本地镜像
  • docker rmi [image] - 删除本地镜像
  • docker run [options] [image] - 运行一个容器
  • docker ps - 列出正在运行的容器
  • docker stop [container] - 停止运行中的容器
  • docker rm [container] - 删除容器
  • docker exec -it [container] [command] - 在运行中的容器中执行命令
  • docker-compose up - 启动 Docker Compose 服务
  • docker-compose down - 停止并删除 Docker Compose 服务
  • docker logs [container] - 查看容器日志
  • docker network ls - 列出 Docker 网络
  • docker volume ls - 列出 Docker 卷
  • docker inspect [container/image] - 查看容器或镜像的详细信息
  • docker build -t [image] [path] - 从 Dockerfile 构建镜像
