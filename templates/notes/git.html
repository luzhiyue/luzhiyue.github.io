Git 备忘录

  • git init - 初始化一个新的 Git 仓库
  • git clone [url] - 克隆一个远程仓库
  • git add [file] - 添加文件到暂存区
  • git commit -m "message" - 提交更改
  • git push - 推送更改到远程仓库
  • git pull - 从远程仓库拉取更改
  • git status - 查看当前工作区状态
  • git branch - 列出、创建或删除分支
  • git checkout [branch] - 切换到指定分支
  • git merge [branch] - 合并指定分支到当前分支
  • git log - 查看提交历史
